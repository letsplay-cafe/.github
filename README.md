# .github

Org-wide GitHub metadata for **Letsplay Cafe**.

- `profile/README.md` — the org profile page shown at [github.com/letsplay-cafe](https://github.com/letsplay-cafe)
- `profile/assets/` — brand assets used by the profile (banner, avatar source, screenshots)

The org avatar is `profile/assets/avatar.png` — upload it at **Org settings → Profile → Profile picture** if it ever needs to be restored.
